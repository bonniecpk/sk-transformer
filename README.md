###Dependencies
- Use Ruby 2.1.3

###Setup
    [host]$ git clone git@github.com:bonniecpk/sk-transformer.git
    [host]$ bundle

###To run
    [host]$ ruby transformer.rb -i /path/to/input -o /path/to/output

###To run tests
    [host]$ rspec

###To see command-line options
    [host]$ ruby transformer.rb
